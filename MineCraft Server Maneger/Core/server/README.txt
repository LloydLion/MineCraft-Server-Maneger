Это jar файлы серверов (без расширения, так было удобнее)
Сюда можно добавлять свои сервера (желательно craftbukkit, bukkit или spigot, но никак не forge или vanila! Всё сломаешь (наверное)), называть файл сервера как название версии
Учти вместе с сервером нужно добавить новую папку в \pers\world\ с названием версии
Сервера можно скачать просто с гугла, "скачать bukkit (ВЕРСИЯ)"
